# Containerum UI
Containerum UI project is Web User Interface for [Containerum](https://github.com/containerum/containerum).

## Prerequisites


## Installation
* Kubernetes

### Using Helm

```
  helm repo add containerum https://charts.containerum.io
  helm repo update
  helm install containerum/ui
```
